# Autonomous Software Factory

## Agency-Grade Monorepo Template
This repository is a **Standardized Infrastructure** for building production-ready software efficiently using Agentic AI. It uses a **Runbook-Driven Development** architecture.

## The Command Center (.system/)
All project intelligence, laws, runbooks, and state are centralized in the `.system/` directory.

### To Start a Project:
1.  Read the master operational manual in **[.system/workflows/WORKFLOWS.md](.system/workflows/WORKFLOWS.md)**.
2.  Populate your requirements in **`.system/prompts/prompt.md`**.
3.  Instruct your AI to: *"Read `.system/workflows/INITIALIZATION.md`"*.

## Key Features
- **Zero-Prompt Architecture**: Instructions are embedded in executable markdown runbooks.
- **One Source of Truth**: All metadata and instructions live in `.system/`.
- **Polyglot Monorepo**: Turborepo + PNPM + UV (Python).
- **Professional Pipeline**: Built-in gates for Testing, Security, and Quality.
- **Autonomous Ready**: Optimized for zero-hallucination agentic development.
